# Solution of an LP
---
>[!definition]
>- **Feasible Point:** a vector $v\in\mathbb R^n$ that satisfies the constraints $Av = b, v\geq0$
>- **Feasible Region:** A set of feasible points.
>- **Optimum:** The feasible point that gives the maximum( or minimum ) cost.
